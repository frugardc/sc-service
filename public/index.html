ASpellchecker.info

Web Service for the GNU Aspell Spellchecking Library

No affiliation with the GNU Project :)

Introduction
ASpellchecker.info is a REST Style Web Service wrapping the GNU Aspell Free and Open Source spell checker. Please see the Project's Website for more information.
Resources
ResourceMethodsOptions
/spellchecker GET,POST
OptionMandatory?DefaultAcceptsDescription
:word true NA string The word that you would like spellchecked
:language false en_US string: 'en_US' (US English) or 'es' (Spanish) currently supported The language of the word you would like spellchecked.
:format false html string [:json,:xml,:html] Format you would like the response in. JSON and XML are preferable to machines, HTML is probably preferable to humans
:suggestions_when_correct false false Boolean: true or false Gives the option to return suggestions for similar words even if the spellchecked word is spelled correct.
